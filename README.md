# Portfolio Website

Persönliche Portfolio-Website auf Basis von Vite, React und Tailwind CSS.

## Lokale Entwicklung

Voraussetzung: Node.js und npm.

```sh
# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev
```

## Build

```sh
npm run build
```

## Technologien

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
